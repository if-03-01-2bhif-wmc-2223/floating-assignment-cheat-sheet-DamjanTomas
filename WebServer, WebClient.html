World Wide Web


The world wide web is such a succes, because it connected the world in a way that made it just easier to get information, share and communicate.

Users can also share their thoughts with for example web sites and blogs.

Recourses are addressed on the world wide web with an URL which stands for Uniform Resource Locator.

The general components of an URL are a schema, also a host, a path and a query string.

Web Server


A web server is a server that uses HTTP which means hypertext transfer protocol to respond to client requests that are made over the world wide web.

It connects to the Internet and and supports physical datas also with other devices that are connected to the world wide web.

Content is stored on the web server on servers using a database. Content is described on webpages with text, links, images audio or videos among other things.

Web Client


A web client refers to the web browser over the user's machine.

A web client is used for connecting to a web server over HTTP.

A web server and a web client aren't the same, a web server refers to a web client over the user's pc.
