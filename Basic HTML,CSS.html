CSS

A css rule consists of a selector and a decleration block. Selector: h1 Decleration: {color(property):blue(Value); font-size:12px;}

direct child(>); descendant(space); adjacent sibling(+); sibling(~);

The descendant selector matches all elements that are descendants of a specified element. The descendant selector is space
The child selector selects all elements that are the children of a specified element. The selector is (>)
The adjacent sibling selector is used to select an element that is directly after another specific element.
Sibling elements must have the same parent element, and "adjacent" means "immediately following". The selector is (+)

tomato = red/ff6347; orange = orange/ffa550; dodgerblue = lightblue/0000ff; mediumseagreen = lightgreen/3cb371; gray = gray/787878; slateblue = purple/6a5acd; violet = pink/ee82ee; lightgray = lightgray/b4b4b4;

Declarations for color, background color:
body{
background-color: lightblue;
}
h1 {color:blue;}

Pseudo classes for the (a) element:
a:link {
color: #FF0000;
}
a:visited {
color: #00FF00;
}
a:hover {
color: #FF00FF;
}
a:active {
color: #0000FF;
}

Declaration for background image:
body{ background-image: url(Test.jpg); }

Declaration for text alignment (left, right, center, justify):
h1{
text-alignment: left;
}
h2{
text-alignment: right;
}
h3{
text-alignment: center;
}
div{
text-alignmnet: justify;
}

Declaration for text decoration (overline, underline, line-through):
h1{
text-decoration-line: overline;
}
h2{
text-decoration-line: underline;
}
h3{
text-decoration-line: line-through;
}

Declaration for text transformation (uppercase, lowercase, capitalize):
p.uppercase{
text-transform: uppercase;
}
p.lowercase{
text-transform: lowercase;
}
p.capitalize{
text-transform: capitalize;
}

On computer screens, sans-serif fonts are easier to read than serif fonts.

}

}

Declarations for font style normal and italic:

}

font-style: italic;
}

Declarations for font sizes:
