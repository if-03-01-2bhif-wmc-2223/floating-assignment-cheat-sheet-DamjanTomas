HTML Lists

- a description list

- a list item

CSS Lists

Make bullet a square
ul{
list-style-type: square;
}
Make bullet an image named "my-bullet.png"
ul{
list-style-image: url('my-bullet.png');
}
Make bullet an uppercase roman number
ol{
list-style-type: upper-roman;
}
Set bullet position to be inside the list item's text flow
ul{
list-style-position: inside;
}
Set bullet position to be outside the list item's text flow
ul{
list-style-position: outside;
}

CSS Tables

Black dotted border of 1 px width.
table{
border: 1px dotted black;
}
Solid border of 1 px width having a red color and rounded corners
table{
border: 1px solid red; border-radius: 5px;
}
Merge the borders of table cells so that they appear as one line
td{
border-collapse: collapse;
}
