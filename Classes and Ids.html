Box Model

• Block level elements always starts on a new line
• Block level element can have customized margin and padding
• Block level element can contain other block level element or Inline-level element
• If the width is not manually set, A block level element always takes up full width of its container
• If the height isn't manually set, Block level element's height is adjusted automatically to fit all the content and other elements within itself.

Classes and Ids

A selector for a certain class:
.class {
css declarations;
}

A selector for a certain html element of a certain class:
element.class {
css declarations;
}

A selector for a certain id:
#id {
css declarations;
}

A selector for a certain html element of a certain id:
#para1 {
text-align: center;
color: red;
}

The start tag of a certain html element with a specific class:
(style)
.country {
background-color: black;
color: white;
padding: 8px;
}
(/style)

The start tag of a certain html element with a specific id:
(style)
#myHeader {
background-color: lightblue;
color: black;
padding: 40px;
text-align: center;
}
(/style)

Do classes select one or more elements?
Multiple HTML elements can share the same class.
Do ids select one or more elements?
They select one element
How often can classes / ids occur on one page?
ids once. Classes don't have a limit.

Floating

Write down the css declaration to make an html element float to the left
p{
float: left;
}

Make an html element stop floating around another (for a specific direction, for all directions)
table, th, td {
border: 0.01em;
border-color: green;
border-style: dashed;
border-radius: 5px;
}
